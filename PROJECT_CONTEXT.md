# Song Zhiduo Portfolio — Project Context

## 1. 项目简介

这是我的个人作品集网站（Personal Portfolio）。

目标不是展示大量技术栈，而是展示我的学习能力、项目实践、内容创作能力以及个人成长。

网站主要用于：

- 求职
- 个人品牌
- 社交平台展示（小红书、播客等）
- 长期记录项目成果

目前采用：

- HTML5
- CSS3
- JavaScript（少量）
- Git
- GitHub
- GitHub Pages
- Tencent CloudBase Hosting

整个项目坚持：

> Simple · Modern · Clean · Maintainable

# 3. 网站设计原则

整个网站遵循：

Modern Minimal Tech

参考风格：

- Apple
- Linear
- Stripe
- Notion

设计原则：

- 大量留白
- 蓝色渐变
- 几何元素
- 不堆技术栈
- 图片优先
- 内容简洁

项目封面：

不放标题。

不放 Logo。

不放文字。

采用抽象视觉表达。

例如：

Portfolio：
数字世界

Podcast：
声音传播

Knowledge：
知识网络

Photography：
视觉观察

# 6. 每次更新应该告诉 AI 的内容

如果以后让 AI 接手，请补充：

① 本次修改目标

例如：

我要增加一个新的项目卡片。

我要修改 Footer。

我要新增 Skills。

---

② 当前目录结构

例如：

assets/

covers/

icons/

images/

index.html

style.css

script.js

---

③ 当前项目截图

最好上传：

首页

项目区域

Footer

移动端

这样 AI 不需要猜。

---

④ 当前代码

如果修改：

HTML

CSS

JavaScript

直接上传完整文件。

不要只发局部。

---

⑤ 修改原则

始终保持：

简洁

现代

统一

易维护

不要引入复杂框架。
